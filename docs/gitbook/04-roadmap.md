# Roadmap

Stabolut is being built in phases. Here's where we are and where we're going.

## Phase 1 — MVP ✅ (Current)

- **Smart contracts:** USB, sUSB, USBVault, AaveV3Strategy, Treasury
- **Audit:** Independent security review completed
- **Testnet:** Live on Arbitrum Sepolia with faucet bot
- **Frontend:** Testnet preview at testnet.stabolut.com
- **Backend:** Ponder indexer powering real-time stats and APY
- **Demo:** Paper trading stack for investor presentations

**Status:** Contracts complete, testnet operational, investor demo ready.

## Phase 2 — Mainnet Launch 🔜

- Deploy contracts to Arbitrum One mainnet
- Initial liquidity pool on Curve / Camelot
- sUSB yield distribution live
- Public dashboard with TVL, APY, proof of reserves
- Launch marketing campaign

**Target:** Q3 2026

## Phase 3 — Multi-Collateral

- Add **BTC** as collateral (delta-neutral hedging via perps)
- Add **ETH** native support beyond Aave
- Expand yield sources (Morpho, Pendle)
- Multi-collateral sUSB with diversified yield

## Phase 4 — Multi-Chain & Growth

- Deploy to additional L2s (Base, Optimism)
- Cross-chain USB (bridged or canonical)
- USB as collateral on lending protocols
- Partnership program for liquidity providers
- Governance token (decentralized control)

## Beyond

- Multi-currency stablecoins (EUR, GBP, etc.)
- Institutional-grade proof of reserves
- Regulated wrapper for compliant jurisdictions
