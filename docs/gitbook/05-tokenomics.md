# Tokenomics & Strategy

## How Yield Works

The USB ecosystem generates yield through a simple, transparent mechanism:

```
User deposits USDC/USDT
        │
        ▼
USBVault supplies to Aave v3
        │
        ▼
Aave generates lending yield
        │
        ▼
90% → sUSB holders (via exchange rate)
10% → Protocol Treasury
```

## Fee Structure

| Fee | Percentage | Recipient | Purpose |
|---|---|---|---|
| Yield share | 10% | Treasury | Protocol operations, audits, growth |
| Mint/Burn | 0% | — | Free to mint and redeem USB |
| sUSB wrap | 0% | — | Free to wrap/unwrap |

## Growth Strategy

**Phase 1 — Testnet & Community**
- Gather beta testers via Telegram faucet
- Collect feedback on UX and contracts
- Build community around yield-bearing narrative

**Phase 2 — Mainnet Liquidity**
- Bootstrap liquidity on Arbitrum DEXs
- Points program for early depositors
- Cap system to manage TVL growth safely

**Phase 3 — Scale**
- Exchange listings (CEX + DEX)
- Partnership with lending protocols
- Multi-collateral expansion
